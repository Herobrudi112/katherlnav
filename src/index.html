Key Features

  • Manual Weight System: Set custom weights for connections or use automatic distance-based weights
  • Dual Applications: Separate editor and viewer for different use cases
  • Path Finding: Dijkstra's algorithm for shortest path calculations
  • Import/Export: Load from rooms.json or custom JSON, export for sharing
  • Interactive Canvas: Drag & drop, zoom, pan, and click interactions
  • Responsive Design: Works on desktop and mobile devices

Workflow

  1. Design: Use the editor to create your navigation graph by adding rooms and connections
  2. Customize: Set manual weights for connections or let the system calculate automatic distances
  3. Test: Verify connectivity and find shortest paths between rooms
  4. Export: Save your graph as JSON for distribution
  5. Share: Distribute the JSON file to users who need navigation
  6. Navigate: Users open the navigator to explore your graph
