How It Works

1

Design

Use the Graph Editor to create your building's navigation structure. Add nodes for rooms, edges for connections, and frames for organization.

2

Configure

Set manual weights for connections to represent distance, difficulty, or preference. Organize spaces with colored frames for better visual clarity.

3

Export

Save your navigation graph as a JSON file. The system automatically includes all nodes, edges, and frame information for complete portability.

4

Navigate

Load your exported graph in the Navigation Viewer to explore paths, find optimal routes, and help users navigate your building efficiently.
